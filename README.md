# Bike Rental Analysis

This repository contains an end-to-end analysis of bike rental patterns using the UCI Bike Sharing Dataset.

## Overview
The project includes:
- Data acquisition and cleaning.
- Exploratory data analysis (EDA) of temporal trends.
- Machine learning modeling using a Random Forest Regressor to predict demand.

- `SRM_Insider_Bike_Rental_Analysis.ipynb`: Main analysis notebook.
